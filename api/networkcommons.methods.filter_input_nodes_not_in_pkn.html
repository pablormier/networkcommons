networkcommons.methods.filter_input_nodes_not_in_pkn

networkcommons.methods.filter_input_nodes_not_in_pkn(data, pkn)

Filters the input nodes in the ‘data’ dictionary that are not present in the PKN.

Parameters:

  • data (dict) – A dictionary containing the input nodes.

  • pkn (nx.DiGraph) – The network object representing the PKN.

Returns:

A new dictionary containing only the input nodes that are present in the PKN.

Return type:

dict
